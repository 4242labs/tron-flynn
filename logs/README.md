# Agent logs

Two kinds, same folder, same discipline: facts with evidence pointers (SHA, PR#) — no narrative.

- **Maintenance** — `log-YYMMDD-HHMM-<slug>.md`: one per maintenance session on the agent itself (persona, skills, install kit).
- **Run** — `log-YYMMDD-HHMM-run-<project>.md`: one per supervision run, written at run end per `skills/skill-session-end.md`. Its self-enhancement section is the backlog for the next maintenance session.
